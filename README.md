# 📚 NEU Library — Visitor Management System
 
A web-based **Visitor Management System** for the Library Department of New Era University.  
Designed to log and manage library visitor records digitally.
 
---
 
## 🌐 Deployed Application
👉(https://neu-library.netlify.app/)
 
## 📁 Repository
👉 [github.com/leiangelyusi-cmd/neu-library](https://github.com/leiangelyusi-cmd/neu-library)
 
---
 
## ✨ Features
- 📋 **Visitor Logging** — Students and guests can log their library visit
- 🔐 **Admin Login** — Staff and faculty can access the admin portal
- 🏫 **NEU Branded** — Built specifically for New Era University's Library Department
 
---
 
## 🚀 How to Use
1. Go to the [Live App](https://neu-library.netlify.app)
2. Click **"Log My Visit"** to record a library visit
3. Staff/Faculty click **"Admin Login"** to access the admin portal
